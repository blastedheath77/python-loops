The For Loop

Let Python do the counting

A for loop automatically steps through a sequence of values, one at a time. You don't need to create a counter, update it, or write a condition — Python handles all of that.

Manual management
  • i = 0 — create the counter
  • while i < 5: — write the condition

Automatic ✓
  • for i in range(5): handles all three
  • Python creates, checks, and updates i
  • You just say what sequence to step through

The for loop syntax

for variable in sequence:
    # body runs for each value

range(5) produces the sequence [0, 1, 2, 3, 4]. On each iteration, i takes the next value.

The for loop is the right choice whenever you have a known sequence to walk through — numbers, list items, characters in a string, and much more.
