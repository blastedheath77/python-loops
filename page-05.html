range() Unpacked

Controlling your counting

range() can take 1, 2, or 3 arguments: range(stop), range(start, stop), or range(start, stop, step). This gives you precise control over which numbers the loop visits.

All three forms

range(stop) → 0, 1, 2, … stop−1
Example: range(5) → 0, 1, 2, 3, 4
range(start, stop) → start, start+1, … stop−1
Example: range(2, 7) → 2, 3, 4, 5, 6
range(start, stop, step) → start, start+step, … < stop
Example: range(0, 10, 2) → 0, 2, 4, 6, 8
Common gotcha: stop is exclusive — the last value in the range is stop - 1 (for positive step). So range(1, 6) gives 1, 2, 3, 4, 5 — not 6.
