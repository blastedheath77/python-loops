Looping Through Lists

One item at a time

for loops aren't limited to numbers. They can step through any sequence — a list of strings, a list of mixed values, even a string character by character. The loop variable takes the value of each item in turn.

Iterating lists directly

When you write for colour in colours:, the variable colour is assigned each list item's value directly — not its index number.

This makes the code read naturally: "for each colour in colours, do something with colour."

Strings are sequences too

for char in "hello":
    print(char)
# prints: h, e, l, l, o

Every character in a string can be visited one by one — because a string is just a sequence of characters.
