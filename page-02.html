The While Loop

Repeat while something is true

A while loop checks a condition before each repetition. If the condition is True, the loop body runs. If False, the loop stops. The condition is checked every single time.

while The keyword that starts the loop.
condition count < 3 — checked before every iteration. Must eventually become False.
body The indented lines that run each time. They must change something that affects the condition.
Infinite loop danger: If the condition never becomes False, the loop runs forever and your program freezes. Always ask: "What will make this condition eventually become False?"
