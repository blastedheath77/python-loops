Break & Continue

Fine-tuning loop flow

break immediately exits the entire loop. continue skips the rest of the current iteration and jumps to the next one. Both give you fine-grained control over what happens inside a loop.

Predict the Output — Quiz

After watching the visualiser above, try this modified code:

numbers = [5, 3, 8, 2, 6]
for num in numbers:
    if num > 7:
        break
    if num % 2 == 0:
        continue
    print(num)

What will be printed?

Animated explanation:

break — exit early

  • Use when searching — stop once found
  • Use to enforce a limit
  • Exits the innermost loop only

continue — skip iteration

  • Use when filtering — ignore some items
  • Keeps the loop running
  • Jumps back to the loop condition / next item

Both break and continue work in while loops as well as for loops.
