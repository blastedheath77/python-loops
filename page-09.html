The five essential patterns

Accumulator Start at 0, add each item. Works for sum, product, count.
Search Loop until you find a match, then break. Return True/False or the item.
Filter Use an if condition to skip items that don't qualify.
Build list Start with an empty list, append() items that meet a condition.
Validation Use while to keep asking until a valid answer is given.
