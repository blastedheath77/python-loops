Choosing the Right Loop

While vs For — a decision guide

Use a for loop when you know in advance how many times to repeat (or you have a sequence to walk through). Use a while loop when the number of repetitions depends on a condition you can't predict.

The decision heuristic

Use for when…

  • You have a list to process
  • You know the exact count (range(n))
  • You're iterating a fixed sequence
  • The end is determined by data, not logic

Use while when…

  • The stop condition is unpredictable
  • You're waiting for user input
  • You need to retry until something works
  • The loop count isn't known upfront

Many problems could use either loop type, but one is almost always cleaner and clearer. When in doubt, if you can easily express the repetition with range() or a list, use for.
